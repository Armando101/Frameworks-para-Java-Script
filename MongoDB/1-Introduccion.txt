
Activar el servicio
	- systemctl start mongo
	- mongo

Crear una base de datos
	Una base de datos necesita mínimo un documento o registro para ser creada. Podemos usar una base de datos con le comando
		- use nombre_db

	Si esta base no tiene nada no será creada, podemos ver las db con el comando:
		- show dbs;

	Puedo crear una colección con el siguiente comando:
		- db.nombre_coleccion.save({atributo: valor})

	Ejemplo:
		- db.usuarios.save({nombre: 'Armando', apellido: 'Rivera'})

	Ahora ya podemos visualizar que la db ha sido creada.
		- show dbs;

	Podemos ver las colecciones
		- db.collections;

	Podemos buscar las colecciones:
		- db.usuarios.find();
